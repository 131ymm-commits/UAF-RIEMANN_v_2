# UAF-RIEMANN: Matter-Antimatter Operator for Riemann Zeta Function

**Автор:** 131ymm-commits  
**Дата создания:** 2026-05-14  
**Лицензия:** MIT (или другая)

## Краткое описание

Этот репозиторий содержит экспериментальный квантово-статистический оператор, построенный на переходах между residue-классами простых чисел. Оператор имеет структуру **материя + антиматерия + нейтральный сектор** с аннигиляционным блоком. Его статистическая сумма `Z(β) = Tr exp(-β H)` приближает дзета-функцию Римана `ζ(1+β)` с ошибкой ~25–35% в диапазоне `β ∈ [2.2, 4.4]`.

## Основные идеи

- Простые числа → «атомы материи» (положительный заряд)
- Отрицательные и нейтральные (модульные) простые → антиматерия и нейтроны
- Переходы между residue-классами (mod 30) с весами, зависящими от gap’а
- Гамильтониан: блочная матрица 3×dim с аннигиляцией между материей и антиматерией

## Лучший достигнутый результат

- **Ошибка:** ~25.6% (среднее относительное отклонение `Z_scaled` от `ζ(1+β)`)
- **Параметры:** annihilation=2.35, neutral=1.55, coupling=0.85, reg=1.45, global_scale=7.1
- **LIMIT=5_000_000, MODULUS=30**

## Структура

- `best_model/` – финальная оптимизированная версия (код + параметры)
- `experiments/` – все проведённые эксперименты с датами, результатами и выводами
- `utils/` – вспомогательные функции для построения матрицы переходов и вычисления Z(β)

## Как использовать

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите `best_model/matter_antimatter_operator_v3.py`
3. Для воспроизведения любого эксперимента см. соответствующий файл в `experiments/`
