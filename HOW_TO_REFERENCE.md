# Как ссылаться на эксперименты этого репозитория в разговоре с ИИ (Claude, Grok, DeepSeek, ChatGPT, Gemini)

Ты можешь просто дать ссылку на репозиторий и сказать:

> "У меня есть репозиторий [ссылка]. Посмотри папку `experiments/`, файл `2026-05-14_1430_matter_antimatter_zeta_25percent.md`. В нём описан эксперимент, где ошибка 25.6%. Возьми эту модель как базовую."

ИИ сможет прочитать markdown-файл с датой, временем, параметрами, результатами и выводами. Формат единообразен для всех экспериментов.

### Формат файла эксперимента
Эксперимент: <название>
Дата: YYYY-MM-DD
Время: HH:MM UTC
Модель: SOM / Matter-Antimatter / Dirichlet-L и т.д.
Параметры: ...
Результат: ошибка X%
Вывод: ...
Код: (ссылка на файл или встроенный)

text

Такой формат понимают все современные LLM.
